API Endpoints Dashboard

Main Routes

  • GET / - Check server status.

User Routes

  • POST /api/users/register - Register a user.
  • POST /api/users/login - User login.
  • GET /api/users/all - Get all users.
  • GET /api/users/private - Test authentication.
  • DELETE /api/users/delete - Delete a user.
  • PUT /api/users/update - Update a user.

Movie Routes

  • GET /api/movies/movies - Get all movies.
  • POST /api/movies/create - Add a new movie.
  • DELETE /api/movies/delete - Delete a movie.
  • PUT /api/movies/update - Update movie details.

Review Routes

  • GET /api/reviews/reviews - Get all reviews.
  • POST /api/reviews/create - Add a new review.
  • DELETE /api/reviews/delete - Delete a review.
  • PUT /api/reviews/update - Update review details.
